# Netflix Data Analysis using Exploratory Data Analysis (EDA)

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset. The main goal of this project is to analyze the dataset, identify trends and patterns, and visualize important insights using Python libraries.

The project helps in understanding:

* Distribution of Movies and TV Shows
* Top content-producing countries
* Ratings distribution
* Release year trends
* Most popular genres

# Results and Observations

* Movies are more than TV Shows on Netflix.
* United States produces the highest content.
* TV-MA is the most common rating.
* Most content was released after 2010.
* Comedy and Drama are among the most popular genres.
